Uni Products

Uni Products is an internal app that helps the sales team of UniTrade company to get through their daily tasks. The has different features and functions that help in maintaining the progress of work and ease the communicating with clients.
App Feautres
  • Host the products owned by the company
    • Product Categories
    • Product Brands
    • Product details
  • Host the clientâ€™s details for each sales person
    • Allow creating new clients
    • Allow editing current clients
  • Generate quotes for current clients
    • Allow creating new quotes for current user clients only
    • Allow to view current quotes
    • Sending quotes through an email serivce

Every sales person is provided with a user name and password by the admin which will allow them to access the normalUser features of the app mentioned above. The sales executive is expected to create his/her clients which are private for each person.

The sales person will be able to supply his/her client needs for prices and technical data from within the application by either accessing the quotation page where a quote with the required items could be arranged for each client, or through accessing the product tabs and share the data sheets for the clients who request a data sheet.

Uni Product app is in a constant progress and will have additional features in the future.
